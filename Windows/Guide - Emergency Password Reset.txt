### Offline Windows password reset instructions using chntpw


1. Boot into linux live USB, lubunut/kubuntu are both fairly lightweight
2. Connect to internet
3. Open terminal and run below commands
    sudo apt-get install chntpw
    sudo mkdir /media/os
    sudo blkid
        note partition ID, e.g. /dev/sda1
    sudo mount /dev/PARTITIONID /media/os
        If can't mount because windows is hybernating etc. run: 
            sudo mount -t ntfs-3g -o remove_hiberfile /dev/PARTITIONID /media/os
        If issue persists, run: 
            sudo ntfsfix /dev/PARTITIONID 
    cd /media/os/Windows/System32/config
    sudo chntpw -i SAM
4. Follow prompts, quit and save when done, reboot
