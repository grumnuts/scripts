Boot into linux live USB, lubunut/kubuntu are both fairly lightweight

Connect to internet

Open terminal

sudo apt-get install chntpw

sudo mkdir /media/os

sudo blkid

note partition ID, e.g. /dev/sda1

sudo mount /dev/PARTITIONID /media/os

If can't mount because windows is hybernating etc.
sudo mount -t ntfs-3g -o remove_hiberfile /dev/PARTITIONID /media/os

If issue persists:
sudo ntfsfix /dev/PARTITIONID 

cd /media/os/Windows/System32/config

sudo chntpw -i SAM

Follow prompts, quit and save when done, reboot
